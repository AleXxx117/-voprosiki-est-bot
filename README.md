# Вопросики есть

Готовая Telegram игра для личного чата, группы обсуждений и канала.

Telegram бот. [@voprosikiestbot](https://t.me/voprosikiestbot).

## Что умеет бот

- Перед игрой показывает меню из шести тем. Психология, быт, отношения, любовь, природа и духовность.
- Предлагает выбрать 5, 10, 15, 20 или 30 раундов.
- Принимает ответы обычными сообщениями и допускает одну небольшую опечатку в длинном слове.
- Сам показывает две подсказки, затем постепенно открывает по одной букве.
- Начисляет от 1 до 5 очков в зависимости от скорости ответа.
- Ведёт личную статистику, серию ответов и рейтинг игроков.
- Имеет кнопки «Пропустить» и «Стоп игра».
- Публикует вопросы в Telegram канал вручную или автоматически по расписанию.
- Содержит 72 проверенных раунда с точными ответами и 1 036 800 комбинаций открытых вопросов без единственного правильного ответа.
- Не сохраняет тексты личных ответов на открытые вопросы.

## Быстрый запуск

Понадобится Node.js 20 или новее.

1. Открой [@BotFather](https://t.me/BotFather), отправь `/newbot`, укажи название `Вопросики есть` и свободное имя, которое заканчивается на `bot`.
2. Скопируй `.env.example` в файл `.env`.
3. Вставь новый токен в `BOT_TOKEN`.
4. Запусти команды.

```bash
npm install
npm start
```

Если токен когда-либо отправлялся в чат или попадал в открытый доступ, не используй его повторно. В BotFather выбери бота, открой `API Token`, отзови старый токен и создай новый.

## Настройка владельца

1. После первого запуска отправь боту команду `/id`.
2. Скопируй свой цифровой Telegram ID в `ADMIN_IDS` внутри `.env`.
3. Перезапусти бота.

После этого станут доступны команды владельца.

```text
/setchannel @имя_канала
/publish
/settimes 10:00,19:00
/autopost_on
/autopost_off
/status
```

Для приватного канала можно опубликовать в нём команду `/bind`. Бот запомнит цифровой ID канала.

## Как подключить к каналу

1. Добавь бота администратором канала и разреши ему публиковать сообщения.
2. Отправь боту в личном чате `/setchannel @имя_канала`.
3. Укажи время, например `/settimes 10:00,19:00`.
4. Включи публикации командой `/autopost_on`.

Участники не могут писать игровые ответы прямо в ленте канала. Для общей игры привяжи к каналу группу обсуждений и добавь бота администратором этой группы. Администратор группы получает обычные сообщения игроков. Другой вариант, отключить Privacy Mode через `/setprivacy` в BotFather и затем заново добавить бота в группу. Это соответствует [официальным правилам получения сообщений Telegram ботами](https://core.telegram.org/bots/faq).

После подключения запусти в группе `/game`. Бот покажет выбор темы и количества раундов.

## Настройки `.env`

```dotenv
BOT_TOKEN=новый_токен_BotFather
ADMIN_IDS=твой_цифровой_ID
CHANNEL_ID=@имя_канала
AUTOPOST_ENABLED=false
AUTOPOST_TIMES=10:00,19:00
TIMEZONE=Europe/Kyiv
ROUND_STEP_SECONDS=20
DATA_FILE=./data/state.json
```

`ROUND_STEP_SECONDS` задаёт интервал между первой подсказкой, второй подсказкой и каждым открытием буквы. Допустимое значение, от 5 до 300 секунд.

## Команды игроков

```text
/start       Главное меню
/game        Начать игру
/stopgame    Закончить текущую игру
/question    Открытый вопрос
/stats       Личная статистика
/top         Рейтинг
/rules       Правила
/help        Все команды
```

## Запуск через Docker

```bash
docker compose up -d --build
```

Статистика хранится в `./data/state.json`. При размещении в облаке подключи постоянный диск к каталогу `/app/data`, чтобы очки сохранялись после повторного развёртывания.

## Размещение на Railway

В проект уже добавлены `Dockerfile` и `railway.json`.

1. Загрузи содержимое проекта в приватный репозиторий GitHub.
2. В Railway создай новый проект из этого репозитория.
3. Добавь переменные из `.env.example` в раздел Variables. Файл `.env` в GitHub не загружай.
4. Для сохранения статистики создай Volume и подключи его к `/app/data`.
5. После запуска открой бота в Telegram и отправь `/start`.

Боту не нужен публичный домен, потому что он получает обновления через безопасный long polling.

## Проверка проекта

```bash
npm run check
```

Команда проверяет механику ответов, подсчёт очков, расписание, отсутствие повторов в генераторе и подтверждает наличие более миллиона комбинаций.
